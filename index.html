Application Development and Emerging Technologies

Course Description

This course provides an opportunity for students to learn new approaches to creating web pages using server-side scripting languages.
It introduces dynamic and interactive tools and methods in web development, focusing on the use of PHP and MySQLi for building interactive and dynamic web pages with database applications.



College: College of Computing and Information Technology
Department: Information Technology and Information Systems Department
Course Title: Application Development and Emerging Technologies
Course Code: IT327
Pre-Requisite: IT225/L Object-Oriented Programming
Co-Requisite: Applications Development and Emerging Technologies
Credit Units: 2 units lecture / 1 unit laboratory
